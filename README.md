# Exploratory-Data-Analysis

Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a retail sales dataset (DMART) to uncover meaningful business insights.
The goal of this project is to analyze sales performance, profitability patterns, customer trends, and category-wise performance using Python-based data analysis tools.

Objectives
Understand overall sales and profit performance
Identify top-performing product categories
Analyze monthly sales trends
Evaluate regional and city-level performance
Study the impact of discount on profit
Generate actionable business insights

Tools & Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Google Colab

Dataset Description
The dataset contains retail transaction data including:
Order ID
Order Date
City / Regio
Category
Sub-Category
Sales
Quantity
Discoun
Profit

Steps Performed
1️.Data Loading
Imported dataset using Pandas
Inspected structure using head(), info(), describe()

2️.Data Cleaning
Checked missing values
Removed duplicates
Converted date column to datetime forma
Created Month and Year columns for trend analysis

3️.Univariate Analysis
Sales distribution
Category distribution
Profit summary

4️.Bivariate Analysis
Sales by Category
Profit by Cit
Monthly Sales Trend
Discount vs Profit relationship

5.Correlation Analysis
Generated heatmap to understand variable relationships

6️.Outlier Detection
Used boxplots to identify extreme values

Key Insights
Certain categories contribute significantly higher revenue.
Sales show seasonal growth patterns across months.
Higher discounts negatively impact profit margins.
Some cities generate high sales but lower profit.
Sales and Quantity show positive correlation.

usiness Recommendations
Optimize discount strategy to protect profit margins.
Focus marketing efforts on high-performing cities.
Promote high-margin product categories.
Plan inventory based on seasonal demand trends.
